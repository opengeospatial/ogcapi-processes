[[req_cwl_deploy_exception-workflow-not-found]]
[requirement]
====
[%metadata]
identifier:: /req/cwl/deploy-exception-workflow-not-found

[.component,class=part]
--
If the `w` parameter has a value and the server cannot find the `w` identifier in the worflows from the body POST request, the status code SHALL be `400`.
--

[.component,class=part]
--
The content of that response SHALL be based upon the schema https://raw.githubusercontent.com/opengeospatial/ogcapi-processes/master/core/openapi/schemas/exception.yaml[exception.yaml].
--

[.component,class=part]
--
The `type` of the exception SHALL be “http://www.opengis.net/def/exceptions/ogcapi-processes-2/1.0/workflow-not-found”.
--

====
