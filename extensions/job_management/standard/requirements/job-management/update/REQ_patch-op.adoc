[[req_job-management_update_patch-op]]
[requirement]
====
[%metadata]
identifier:: /req/job-management/update-patch-op

[.component,class=part]
--
For every created jobs (path `/jobs/{jobID}`), the server SHALL support the HTTP PATCH operation.
--

[.component,class=part]
--
The parameter `jobID` is each `id` property in the jobs list response (JSONPath: `$.jobs[*].id`).
--
