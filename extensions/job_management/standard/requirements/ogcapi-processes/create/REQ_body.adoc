[[req_ogcapi-processes_create_body]]
[requirement]
====
[%metadata]
identifier:: /req/ogcapi-processes/create-body

[.component,class=part]
--
The body of the POST request SHALL be based upon the OpenAPI 3.0 schema https://github.com/opengeospatial/ogcapi-processes/blob/master/openapi/schemas/processes-workflows/execute-workflows.yaml[execute-workflows.yaml]
--

[.component,class=part]
--
The media type `application/json` SHALL be used to indicate that request body contains a processes description encoded as an <<rc_ogcapi-processes,OGC API - Processes>>.
--
