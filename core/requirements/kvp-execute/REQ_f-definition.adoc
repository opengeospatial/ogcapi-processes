[[req_kvp-execute_f-definition]]
[requirement]
====
[%metadata]
identifier:: /req/kvp-execute/f-definition

The operation SHALL support a response parameter `response[f]` with the following characteristics:

[source,YAML]
----
name: response
in: query
required: false
schema:
  type: object
  required:
    - f
  properties:
    f:
      type: string
      format: uri
style: deepObject
explode: true
----
====
