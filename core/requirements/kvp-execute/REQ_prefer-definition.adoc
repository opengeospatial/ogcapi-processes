[[req_kvp-execute_prefer-definition]]
[requirement]
====
[%metadata]
identifier:: /req/kvp-execute/prefer-definition

The operation SHALL support a parameter `response[prefer]` with the following characteristics:

[source,YAML]
----
name: response
in: query
required: false
schema:
  type: object
  required:
    - prefer
  properties:
    prefer:
      type: string
      format: uri
style: deepObject
explode: true
----
====
