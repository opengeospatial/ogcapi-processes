[[req_core_job-results-exception_no-such-job]]
[requirement]
====
[%metadata]
identifier:: /req/core/job-results-exception-no-such-job
label:: /req/core/job-results-exception-no-such-job

If the endpoint is requested using an invalid job identifier, the response SHALL have HTTP status code 404.
The content of that response SHALL be based upon the OpenAPI
3.0 schema https://raw.githubusercontent.com/opengeospatial/ogcapi-processes/master/openapi/schemas/common-core/exception.yaml[exception.yaml].
The `type` of the exception SHALL be "http://www.opengis.net/def/exceptions/ogcapi-processes-1/1.0/no-such-job".
====
