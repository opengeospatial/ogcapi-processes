[[req_ogc-process-description_input-mixed-type]]
[requirement]
====
[%metadata]
type:: general
label:: /req/ogc-process-description/input-mixed-type

part:: An input that can be of mixed type SHALL be defined using the https://tools.ietf.org/html/draft-bhutton-json-schema-00#section-10.2.1.3[`oneOf`] JSON Schema keyword.

part:: Each sub-schema SHALL be a JSON fragment that validates according to the JSON Schema: https://raw.githubusercontent.com/opengeospatial/ogcapi-processes/master/core/openapi/schemas/schema.yaml[schema.yaml].

part:: The first sub-schema in the https://tools.ietf.org/html/draft-bhutton-json-schema-00#section-10.2.1.3[`oneOf`] array SHALL be considered the default format.
====
